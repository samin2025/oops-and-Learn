توی این تمرین قراره یه قدم جلوتر از تمرین قبلی برم و افزودن تقویم و to do list رو یاد بگیرم.

«وقتی تلاش می‌کنیم تقویم فارسی را در صفحه وب از طریق iframe نمایش دهیم، مرورگر خطای ۴۰۴ یا عدم بارگذاری محتوا نشان می‌دهد. علت اصلی این است که سایت منبع یا اجازه‌ی بارگذاری در iframe را نمی‌دهد (Cross-Origin Resource Sharing / X-Frame-Options محدود شده) و یا مسیر URL تغییر کرده است. بنابراین محتوا مستقیماً قابل نمایش در صفحه ما نیست و نیاز به راه‌حل جایگزین، مثل استفاده از API رسمی سایت یا embed کردن یک ویجت دیگر، داریم.»

دوست داشتی ،می‌تونی to do list رو تستش کنی!
